R05 — Lectura con read

Ejercicios Resueltos · Scripting en Bash
Objetivo
Crear un script que pregunte al usuario su nombre y edad, y muestre un mensaje personalizado.
Pasos
4 pasos para completar el ejercicio.
Resultado
El script pide datos y muestra "Me llamo [nombre] y tengo [edad] anios".
Conceptos
Comando read, variables propias, echo.
Pasos
1

Crea el archivo del script

Escribe esto:
$ nano pregunta.sh
2

Escribe el script

Dentro de nano, escribe estas líneas:
Contenido del script:
#!/bin/bash
echo -n "Como te llamas? "
read nombre
echo -n "Cuantos anios tienes? "
read edad
echo "Me llamo $nombre y tengo $edad anios"
¿Qué hace read?
read nombre → Espera a que el usuario escriba algo y pulsa Enter. Lo que escriba se guarda en la variable $nombre.
A diferencia de los argumentos ($1, $2), aquí el usuario introduce los datos durante la ejecución del script.
Guarda y sal: Ctrl+O, Enter, Ctrl+X.
3

Da permisos y ejecuta

Escribe estos dos comandos:
$ chmod +x pregunta.sh
$ ./pregunta.sh
El script te preguntará el nombre y la edad. Escríbelos y pulsa Enter tras cada uno.
4

Comprueba el resultado

Resultado esperado
Como te llamas? Raul
Cuantos anios tienes? 30
Me llamo Raul y tengo 30 anios
Nota: lo que aparece en negrita es lo que tú escribes. El resto lo muestra el script.
