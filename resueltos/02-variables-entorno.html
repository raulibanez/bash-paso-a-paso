R02 — Variables del entorno

Ejercicios Resueltos · Scripting en Bash
Objetivo
Crear un script que muestre información del sistema usando variables de entorno.
Pasos
4 pasos para completar el ejercicio.
Resultado
El script muestra tu usuario, directorio home, carpeta actual y shell.
Conceptos
Variables $USER, $HOME, $PWD, $SHELL.
Pasos
1

Crea el archivo del script

Escribe esto:
$ nano entorno.sh
Primera vez con nano? Revisa los pasos detallados del Ejercicio R01 donde se explica como guardar y salir.
2

Escribe el script

Dentro de nano, escribe estas 5 líneas:
Contenido del script:
#!/bin/bash
echo "Tu usuario es: $USER"
echo "Tu directorio home es: $HOME"
echo "Estas en la carpeta: $PWD"
echo "Tu shell es: $SHELL"
¿Qué son las variables de entorno?
Son variables que el sistema define automáticamente. Empiezan por $ y contienen información como tu nombre de usuario, tu carpeta personal, etc. Puedes verlas todas con el comando env.
Guarda y sal: Ctrl+O, Enter, Ctrl+X.
3

Da permisos y ejecuta

Escribe estos dos comandos:
$ chmod +x entorno.sh
$ ./entorno.sh
4

Comprueba el resultado

Resultado esperado
Tu usuario es: vboxuser
Tu directorio home es: /home/vboxuser
Estas en la carpeta: /home/vboxuser
Tu shell es: /bin/bash
Nota: los valores que aparezcan dependerán de tu sistema. Lo importante es que se muestren las 4 líneas con información real.
Practica por tu cuenta: intenta hacer el Ejercicio P02 — Informacion del sistema, que usa los mismos conceptos.
