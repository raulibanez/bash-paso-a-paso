R01 — Hola Mundo

Ejercicios Resueltos · Scripting en Bash
Objetivo
Crear tu primer script en Bash que muestre "Hola Mundo" por pantalla.
Pasos
5 pasos para completar el ejercicio.
Resultado
Al ejecutar el script aparece el texto "Hola Mundo" en el terminal.
Conceptos
Shebang #!/bin/bash, comando echo, permisos con chmod.
Pasos
1

Abre el terminal

Abre una ventana de terminal en tu sistema Linux.
2

Crea el archivo del script

Usa el editor nano para crear un nuevo archivo:
Escribe esto:
$ nano hola.sh
3

Escribe el script

Dentro de nano, escribe estas 2 líneas:
Contenido del script:
#!/bin/bash
echo "Hola Mundo"
¿Qué hace cada línea?
#!/bin/bash → Le dice al sistema que use Bash para ejecutar el script
echo "Hola Mundo" → Muestra el texto "Hola Mundo" por pantalla
Guarda y sal de nano:
Pulsa estas teclas:
Ctrl + O → pulsa Enter para guardar
Ctrl + X → para salir de nano
4

Da permisos de ejecución

Para poder ejecutar el script, necesitas darle permisos:
Escribe esto:
$ chmod +x hola.sh
Recuerda: sin este paso, el script no se puede ejecutar.
5

Ejecuta el script

Escribe esto:
$ ./hola.sh
Resultado esperado
Hola Mundo
